正好刚听学长讲了计算几何，就收录下一些相关操作吧

图形存储

  1. 点：我们可以直接存储x,y坐标或x,y,z坐标

  2. 向量：起点在原点的向量用x,y坐标或x,y,z坐标表示，任意一个向量可以用两个向量相减的形式表示

Code

  1. 直线：用两个点或一个点和一个向量的形式

基础计算

后面的$\vec{a},\vec{b}$均为$\vec{a}=(x_1,y_1),\vec{b}=(x_2,y_2)$，直线$l_1,l_2$均为$p_1+t\vec{v_1},p_2+t\vec{v_2}$

  1. 两点之间距离：$dist=\sqrt{(x_1-x_2)^2+(y_1-y_2)^2}$

Code

  1. 向量加、减、数乘、数除：x,y对应操作就行了

**Code

  1. 点乘：$dot(\vec{a},\vec{b})=x_1\cdot x_2+y_1\cdot y_2=|\vec{a}|\cdot|\vec{b}|\cdot \cos\left \langle\vec{a},\vec{b}\right \rangle$

Code

  1. 叉积：$cross(\vec{a},\vec{b})=x_1\cdot y_2-x_2\cdot y_1=\cdot \sin\left \langle\vec{a},\vec{b}\right \rangle$

  1. 向量旋转$\alpha$弧度：$x’=x\cos\alpha-y\sin\alpha,y’=x\sin\alpha+y\cos\alpha$

  2. 直线交点：由方程算出交点在任何一条直线上的$t$，再算交点，设$\vec{u}=p_1-p_2,t_1=\frac{cross(\vec{v_1},\vec{u})}{cross(\vec{v_1},\vec{v_2})}$，交点即为$p_1+t_1\vec{v_1}$

  3. 点到直线的距离：用等积法求高
